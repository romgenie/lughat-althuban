# لغة الثعبان (lughat-althuban)

لغة الثعبان هي لهجة برمجة مبنية بالكامل باللغة العربية، تتيح للمطورين كتابة شفرات برمجية (Code) باستخدام كلمات مفتاحية ودوال مدمجة ووحدات قياسية عربية بحتة. يعتمد المشروع على محرك لغة بايثون (Python) القياسي، مما يضمن توافقاً كاملاً مع بيئة بايثون مع توفير تجربة برمجة أصيلة للناطقين بالعربية، دون الحاجة للتعامل مع المصطلحات الإنجليزية.

## نظرة سريعة

إليك مثال يوضح كيفية كتابة برنامج متكامل يحتوي على دالة، حلقة تكرار، معالجة استثناءات، واستيراد وحدة:

```python
استورد رياضيات

دالة احسب_الجذور(الارقام):
    نتائج = قائمه()
    لكل رقم في الارقام:
        حاول:
            اذا رقم < 0:
                ارفع خطا_قيمه("لا يمكن حساب الجذر لرقم سالب")
            نتائج.اضف(رياضيات.جذر(رقم))
        استثني خطا_قيمه:
            اطبع("تم تخطي رقم غير صالح")
        اخيرا:
            اطبع("انتهت محاولة الحساب")
    ارجع نتائج

ارقامي = [9, 16, -4, 25]
اطبع(احسب_الجذور(ارقامي))
```

## التثبيت

تأكد من وجود بيئة بايثون، ثم نفذ الأوامر التالية في سطر الأوامر:

```bash
git clone https://github.com/GalaxyRuler/lughat-althuban.git
cd lughat-althuban
pip install -e .
```

## طرق التشغيل

تدعم لغة الثعبان أربع طرق أساسية لتشغيل الشفرات البرمجية:

1. **تشغيل ملف**: `ثعبان ملف.apy` (تشغيل ملف يحمل امتداد لغة الثعبان).
2. **تشغيل أمر مباشر**: `ثعبان -c 'اطبع("مرحبا بالعالم")'`
3. **الواجهة التفاعلية**: `ثعبان` (للدخول إلى واجهة سطر الأوامر التفاعلية REPL).
4. **عبر استيراد الوحدة في بايثون**: تفعيل بيئة الثعبان من داخل مشاريع بايثون القياسية.

## فهرس الوحدات المتاحة

يدعم المشروع العديد من الوحدات القياسية والأدوات الخارجية (SDKs):

| الاسم العربي | الوحدة الأصلية | مثال على الاستخدام |
| --- | --- | --- |
| نظام_تشغيل | (os) | `نظام_تشغيل.اعرض_بيئة()` |
| مسار_مكتبه | (pathlib) | `مسار_مكتبه.مسار("ملف.txt")` |
| نظام | (sys) | `نظام.مدخلات` |
| رياضيات | (math) | `رياضيات.جذر(16)` |
| اتزامن | (asyncio) | `اتزامن.شغل(دالتي())` |
| جيسون | (json) | `جيسون.نص(قاموسي)` |
| قاعدة_بيانات | (sqlite3) | `قاعدة_بيانات.اتصل("بيانات.db")` |
| طلبات | (requests) | `طلبات.اجلب("http://...")` |
| قارورة | (flask) | `قارورة.تطبيق(__اسم__)` |
