## حزمة التزامن (ب-037)

### وحدة اتزامن (asyncio)
لإدارة المهام المتزامنة وغير المتزامنة في البرنامج.

| الاسم العربي | الوظيفة | مثال استخدام بسيط |
| --- | --- | --- |
| شغل | تشغيل مهمة غير متزامنة | `اتزامن.شغل(مهمتي())` |
| اجمع_مهام | تشغيل عدة مهام | `اتزامن.اجمع_مهام(م1, م2)` |

```python
استورد اتزامن

غير_متزامن دالة مهمة_سريعة():
    انتظر اتزامن.نم(1)
    ارجع "تمت المهمة"

غير_متزامن دالة رئيسية():
    نتائج = انتظر اتزامن.اجمع_مهام(مهمة_سريعة(), مهمة_سريعة())
    اطبع(نتائج)

اتزامن.شغل(رئيسية())
```

### وحدة قارورة (flask)
لبناء تطبيقات الويب وخوادم الواجهات البرمجية.

| الاسم العربي | الوظيفة | مثال استخدام بسيط |
| --- | --- | --- |
| تطبيق_فلاسك | إنشاء تطبيق ويب | `قارورة.تطبيق_فلاسك(__اسم__)` |
| مسار | تحديد مسار رابط | `@تطبيق.مسار("/")` |

```python
استورد قارورة
تطبيق = قارورة.تطبيق_فلاسك(__اسم__)

@تطبيق.مسار("/")
دالة رئيسية():
    ارجع "مرحبا بك في موقعي!"

اذا __اسم__ == "__main__":
    تطبيق.شغل()
```
