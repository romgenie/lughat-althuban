## ١. مجموع قائمة أعداد

**الهدف**: حساب مجموع قائمة أعداد بطريقتين، باستعمال البنائيّ `sum`
وبحلقة صريحة.

```apy
الأعداد = [3, 8, 1, 4, 9, 2]

# الطريقة الأولى: البنائيّ
المجموع_السريع = sum(الأعداد)
اطبع("بالبنائيّ:", المجموع_السريع)

# الطريقة الثانية: حلقة صريحة
المجموع_اليدوي = 0
لكل عدد في الأعداد:
    المجموع_اليدوي = المجموع_اليدوي + عدد
اطبع("باليد:", المجموع_اليدوي)

أكد المجموع_السريع == المجموع_اليدوي
```

## ٢. تصفية الأعداد الزوجية وتربيعها

**الهدف**: استخراج الأعداد الزوجية من قائمة، ثم تربيعها.

```apy
الأعداد = نطاق(1, 11)

الأزواج_المربّعة = [س * س لكل س في الأعداد إذا س % 2 == 0]
اطبع(الأزواج_المربّعة)

أكد الأزواج_المربّعة == [4, 16, 36, 64, 100]
```

## ٣. تكرار الكلمات في نصّ

**الهدف**: حساب تكرار كل كلمة في نصّ قصير، باستعمال `Counter` عبر اسمه
العربي `عداد` من وحدة `مجموعات`.

```apy
استورد مجموعات

النصّ = "تفاحة موزة تفاحة عنب موزة تفاحة"
الكلمات = النصّ.split()

العدّاد = مجموعات.عداد(الكلمات)
اطبع(العدّاد)

أكد العدّاد["تفاحة"] == 3
أكد العدّاد["موزة"] == 2
أكد العدّاد["عنب"] == 1
```

## ٤. تحويل JSON من نصّ إلى قاموس

**الهدف**: قراءة سلسلة JSON وتحويلها إلى قاموس بايثون باستعمال وحدة
`جيسون`.

```apy
استورد جيسون

النصّ_جيسون = '{"اسم": "أحمد", "عمر": 30, "مدن": ["الرياض", "جدة"]}'

البيانات = جيسون.من_نص(النصّ_جيسون)
اطبع(البيانات)
اطبع("الاسم:", البيانات["اسم"])
اطبع("أوّل مدينة:", البيانات["مدن"][0])

أكد البيانات["عمر"] == 30
أكد طول(البيانات["مدن"]) == 2
```

## ٥. إحصائيات سريعة عبر وحدة `احصاء`

**الهدف**: حساب الوسط الحسابي والوسيط والانحراف المعياري لمجموعة قيم.

```apy
استورد احصاء

العلامات = [85, 72, 91, 65, 78, 88, 95, 70]

اطبع("الوسط:", احصاء.وسط(العلامات))
اطبع("الوسيط:", احصاء.وسيط(العلامات))
اطبع("الانحراف:", round(احصاء.انحراف(العلامات), 2))

أكد احصاء.وسيط(العلامات) == 81.5
```

## ٦. حساب فرق زمني بين تاريخين

**الهدف**: قياس عدد الأيام بين تاريخين باستعمال وحدة `مكتبة_تاريخ`.

```apy
استورد مكتبة_تاريخ

البداية = مكتبة_تاريخ.تاريخ(2024, 1, 1)
النهاية = مكتبة_تاريخ.تاريخ(2024, 12, 31)

الفرق = النهاية - البداية
اطبع("عدد الأيام:", الفرق.days)

أكد الفرق.days == 365
```

## ٧. مجموع الأعداد الفردية بدالّة مولِّدة

**الهدف**: استعمال `سلم` (yield) لإنشاء مولِّد يُنتج الأعداد الفردية،
ثم جمعها.

```apy
دالة فرديات(حد):
    """مولِّد يُنتج الأعداد الفردية حتى الحدّ المعطى."""
    لكل عدد في نطاق(1, حد):
        إذا عدد % 2 != 0:
            سلم عدد

الأعداد = list(فرديات(10))
اطبع("الفرديات:", الأعداد)

المجموع = sum(فرديات(10))
اطبع("المجموع:", المجموع)

أكد الأعداد == [1, 3, 5, 7, 9]
أكد المجموع == 25
```

## ٨. صنف بسيط مع وراثة

**الهدف**: تعريف صنف أصل وصنف فرع، واستدعاء التابع المُعاد تعريفه عبر
`الاصل()` (super).

```apy
صنف حيوان:
    دالة __init__(الذات, اسم):
        الذات.اسم = اسم

    دالة يقدّم(الذات):
        ارجع f"أنا {الذات.اسم}."

صنف قطّة(حيوان):
    دالة يقدّم(الذات):
        مقدّمة_الأصل = الاصل().يقدّم()
        ارجع f"{مقدّمة_الأصل} مياو."

مشمشة = قطّة("مشمشة")
اطبع(مشمشة.يقدّم())

أكد مشمشة.يقدّم() == "أنا مشمشة. مياو."
```

## ٩. التعامل مع أخطاء القيم

**الهدف**: التقاط `ValueError` عند تحويل نصّ غير صالح إلى عدد.

```apy
دالة الى_عدد_آمن(نصّ):
    """حوّل النصّ إلى عدد صحيح، أو أعد لاشيء عند الفشل."""
    حاول:
        ارجع int(نصّ)
    استثناء خطا_قيمه:
        ارجع لاشيء

اطبع(الى_عدد_آمن("42"))
اطبع(الى_عدد_آمن("سبعة"))
اطبع(الى_عدد_آمن("3.14"))

أكد الى_عدد_آمن("42") == 42
أكد الى_عدد_آمن("سبعة") == لاشيء
```

## ١٠. تجميع عناصر بقاموس تلقائي

**الهدف**: تجميع كلمات بحسب طولها باستعمال `defaultdict` عبر اسمه
العربي `قاموس_تلقائي`.

```apy
استورد مجموعات

الكلمات = ["شمس", "قمر", "نجمة", "كوكب", "غيمة", "ريح", "برق"]

التجميع = مجموعات.قاموس_تلقائي(list)
لكل كلمة في الكلمات:
    التجميع[طول(كلمة)].append(كلمة)

# اطبع المجموعات بترتيب طول المفتاح
لكل الطول في sorted(التجميع.keys()):
    اطبع(f"طول {الطول}: {التجميع[الطول]}")

أكد set(التجميع[3]) == {"شمس", "قمر", "ريح", "برق"}
أكد set(التجميع[4]) == {"نجمة", "كوكب", "غيمة"}
```
