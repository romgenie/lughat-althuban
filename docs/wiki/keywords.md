## الكلمات المفتاحية في السياق

### الدوال والأصناف

```python
صنف حيوان:
    دالة __init__(ذات, اسم, عمر):
        ذات.اسم = اسم
        ذات.عمر = عمر
    
    دالة تعريف(ذات):
        ارجع f"أنا {ذات.اسم}، عمري {ذات.عمر}"

صنف كلب(حيوان):
    دالة نبح(ذات):
        اطبع("هاو هاو!")

كلبي = كلب("ريكس", 3)
اطبع(كلبي.تعريف())
```

### التحكم في التدفق

```python
لكل رقم في مدى(1, 11):
    اذا رقم % 2 == 0:
        اطبع(f"{رقم} زوجي")
    وإلا اذا رقم % 3 == 0:
        اطبع(f"{رقم} مضاعف 3")
    وإلا:
        اطبع(رقم)
```

### معالجة الاستثناءات

```python
حاول:
    نتيجة = 10 / 0
استثني خطا_القسمه_على_صفر:
    اطبع("لا يمكن القسمة على صفر!")
اخيرا:
    اطبع("انتهت العملية")
```

### الدوال غير المتزامنة

```python
استورد اتزامن

غير_متزامن دالة جلب_بيانات(رابط):
    انتظر اتزامن.sleep(1)
    ارجع {"بيانات": "نتيجة"}

اتزامن.شغل(جلب_بيانات("https://example.com"))
```

### المطابقة الهيكلية (Python 3.10+)

```python
طابق الأمر:
    حالة "بدء":
        اطبع("بدء التشغيل...")
    حالة "إيقاف":
        اطبع("جارٍ الإيقاف...")
    حالة _:
        اطبع("أمر غير معروف")
```
