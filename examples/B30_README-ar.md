# B-030 — مثال: المشي في شجرة الملفات

يوضّح هذا المثال استخدام وحدات المكتبة القياسية بأسماء عربية
(`نظام_تشغيل`، `نظام`، `مسار_مكتبه`) المُضافة في الحزمة B-030.

## التشغيل

```bash
ثعبان examples/B30_filesystem_walk.apy .
```

مثال على الخرج:

```
عدد_الملفات: 142
```

## الخلاصة التقنية

| الاسم العربي     | الوحدة الإنجليزية | الملاحظة                                     |
|------------------|-------------------|----------------------------------------------|
| `نظام_تشغيل`    | `os`              | عمليات نظام التشغيل                          |
| `نظام`           | `sys`             | معلومات المُفسِّر والوسائط والمسارات         |
| `مسار_مكتبه`    | `pathlib`         | مسارات الملفات الكائنية (Python 3.4+)        |

تُستدعى توابع `pathlib.Path` كتوابع حرة مع تمرير الكائن صراحةً:

```python
# العربي:
مسار_مكتبه.موجود(م)      # ≡  pathlib.Path.exists(م)
مسار_مكتبه.اقرا_نص(م)    # ≡  pathlib.Path.read_text(م)
```
