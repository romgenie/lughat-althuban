# الأمثلة التعليمية

سبعة برامج تعليمية تصاعدية مكتوبة بلهجة apython العربية. كل مثال مستقل وقابل للتشغيل مباشرة.

---

## قائمة الأمثلة

| الملف | الموضوع | ما ستتعلمه |
|-------|---------|------------|
| [`01_hello.apy`](01_hello.apy) | مرحبا بالعالم | `اطبع`، الملف الأول |
| [`02_arithmetic.apy`](02_arithmetic.apy) | الحساب والمتغيرات | المتغيرات، العمليات الحسابية، النصوص المُدرَجة |
| [`03_control_flow.apy`](03_control_flow.apy) | تدفق التحكم | `إذا`، `وإلا`، `لكل`، `نطاق` |
| [`04_functions.apy`](04_functions.apy) | الدوال | `دالة`، `ارجع`، القيم الافتراضية |
| [`05_data_structures.apy`](05_data_structures.apy) | هياكل البيانات | القوائم، القواميس |
| [`06_classes.apy`](06_classes.apy) | الأصناف | `صنف`، `__init__`، `الذات`، الدوال |
| [`07_imports.apy`](07_imports.apy) | الاستيراد | `استورد`، `كـ`، الوحدات العربية |

## تشغيل الأمثلة

**تشغيل مثال واحد:**
```bash
apython examples/01_hello.apy
apython examples/06_classes.apy
```

**تشغيل جميع الأمثلة دفعة واحدة (bash/zsh):**
```bash
for f in examples/0*.apy; do
    echo "=== $f ==="
    apython "$f"
done
```

**تشغيل جميع الأمثلة على Windows (PowerShell):**
```powershell
Get-ChildItem examples\0*.apy | ForEach-Object {
    Write-Host "=== $_ ===" -ForegroundColor Cyan
    apython $_
}
```

## شرح كل مثال

### 01 — مرحبا بالعالم

```python
# مرحبا — البرنامج الأول
اطبع("مرحبا، يا عالم")
```

نقطة البداية: ملف `.apy` مكوّن من سطر واحد. `اطبع` تُترجَم إلى `print`.

---

### 02 — الحساب والمتغيرات

```python
# الحساب والمتغيرات
العمر = 25
السنوات_حتى_التقاعد = 65 - العمر
اطبع(f"باقي {السنوات_حتى_التقاعد} سنة")
```

أسماء المتغيرات العربية مدعومة بالكامل. النصوص المُدرَجة (`f"..."`) تعمل داخل الشرطات المزدوجة كما في بايثون العادية.

---

### 03 — تدفق التحكم

```python
# تدفق التحكم: لكل، إذا، وإلا
لكل عدد في نطاق(1, 6):
    إذا عدد % 2 == 0:
        اطبع(f"{عدد}: زوجي")
    وإلا:
        اطبع(f"{عدد}: فردي")
```

**الكلمات المفتاحية المستخدمة:**

| عربي | بايثون |
|-------|--------|
| `لكل` | `for` |
| `في` | `in` |
| `نطاق` | `range` |
| `إذا` | `if` |
| `وإلا` | `else` |

---

### 04 — الدوال

```python
# الدوال: تعريف، استدعاء، قيم افتراضية
دالة جمع(أ, ب=10):
    ارجع أ + ب

اطبع(جمع(5))
اطبع(جمع(5, 7))
```

`دالة` تُترجَم إلى `def`، و`ارجع` إلى `return`. القيم الافتراضية للمعاملات تعمل كالمعتاد.

---

### 05 — هياكل البيانات

```python
# القوائم والقواميس
الفواكه = ["تفاح", "موز", "برتقال"]
الأسعار = {"تفاح": 3, "موز": 2, "برتقال": 4}

لكل فاكهة في الفواكه:
    اطبع(f"{فاكهة}: {الأسعار[فاكهة]} ريال")
```

القوائم (`list`) والقواميس (`dict`) تستخدم البنية التركيبية ذاتها من بايثون. الفهرسة والتكرار يعملان بالكامل.

---

### 06 — الأصناف

```python
# الأصناف: تعريف صنف ودالة بانية
صنف نقطة:
    دالة __init__(الذات, س, ص):
        الذات.س = س
        الذات.ص = ص

    دالة المسافة_من_الأصل(الذات):
        ارجع (الذات.س ** 2 + الذات.ص ** 2) ** 0.5

نقطة_1 = نقطة(3, 4)
اطبع(نقطة_1.المسافة_من_الأصل())
```

**ملاحظات:**
- `صنف` تُترجَم إلى `class`.
- `الذات` تُترجَم إلى `self` (الاتفاقية المعتادة في بايثون).
- أسماء الدوال الخاصة كـ `__init__` تبقى كما هي.
- خصائص الكائن (`الذات.س`) وأسماء الدوال (`المسافة_من_الأصل`) تُطبَّع وفق ADR 0004.

---

### 07 — الاستيراد

```python
# main: استيراد وحدة .apy أخرى عبر import hook
استورد helper كـ مساعد

اطبع(مساعد.مربع(5))
اطبع(مساعد.مكعب(3))
```

```python
# helper.apy
دالة مربع(س):
    ارجع س ** 2

دالة مكعب(س):
    ارجع س ** 3
```

`استورد` تُترجَم إلى `import`، و`كـ` تُترجَم إلى `as`. يتيح خطّاف الاستيراد المُثبَّت في `sys.meta_path` استيراد ملفات `.apy` من ملفات `.apy` أو `.py` أخرى.
